📘 Descripción general del sistema

- Este proyecto corresponde a la evaluación sumativa 2 de la asignatura Desarrollo Orientado a Objetos I.

- El proyecto fue creado a partir de una problemática en una agencia de turismo, en este se aplico el uso de Arraylist para almacenar datos, separar datos mediante .split, el uso de bucles para filtrar datos y recorrer la lista, encapsulamiento de atributos, clases con dependencia.

🧱 Estructura general del proyecto

📁 src/

- ├── app/ # Clase principal con el método Main
- ├── model/ # Clases de dominio (Producto, Encargado)
- ├── service/ # Clase para gestionar datos del archivo .txt (GestionDeDatos)
- └── / # Archivo tours.txt para almacenar datos (Fuera de la carpeta "src")
⚙️ Instrucciones para clonar y ejecutar el proyecto

- Abrir el proyecto en IntelliJ IDEA.

- Ejecuta el archivo Main.java que se encuentra en el paquete app.
